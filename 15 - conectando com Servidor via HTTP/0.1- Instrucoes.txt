Caso o código que abrir não esteja inicializando, verificar se precisa instalar os pacotes dode, para isso utilizar o comando: 
* npm install
no capitulo 2 é preciso instalar o boodstrap vue, usando o comando: 
* npm install bootstrap-vue bootstrap

No capitulo 3 é preciso instalar a dependência axios para fazer funcionar, usando o comando: 
npm i --save axios
